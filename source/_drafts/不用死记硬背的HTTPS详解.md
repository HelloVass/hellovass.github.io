## HTTPS

全称：HTTP Secure or HTTP over SSL or HTTP over TLS

SSL: Secure Socket Layer

TLS: Transport Layer Secure

定义：在 HTTP 之下增加一个安全层，用于保障 HTTP 的保密传输

本质：在客户端和服务器之间用非对称加密协商出一套对称密钥，每次发送消息之前将内容加密，收到之后解密，达到内容的加密传输
